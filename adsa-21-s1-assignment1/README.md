# Assignment 1 — Karatsuba 大数乘法

## 题目描述

实现任意进制下的大整数**加法**与**乘法**（Karatsuba 算法）。

## 算法

- **加法 `add`**：逐位相加，处理进位，支持负数（转调用减法）
- **减法 `sub`**：逐位相减，处理借位，自动判断正负
- **乘法 `KaratsubaMult`**：Karatsuba 分治算法
  - 将 n 位数拆成高位 `a1`、低位 `a0`
  - 递归计算 p0=a0\*b0、p2=a1\*b1、p1=(a1+a0)\*(b1+b0)
  - 结果 = p2\*B^(2k) + (p1-p2-p0)\*B^k + p0
  - 时间复杂度：O(n^1.585)，优于朴素 O(n^2)

## 输入格式

```
<数字1> <数字2> <进制b>
```

## 输出格式

```
<加法结果> <乘法结果>
```

## 示例

```
输入: 123 456 10
输出: 579 56088

输入: 1010 1101 2
输出: 10111 10000010
```

## 编译与运行

```bash
g++ -o a1 main.cpp
echo "123 456 10" | ./a1
```

## 已知问题

- 当其中一个操作数为 `0` 时，乘法返回空字符串而非 `0`
  - 原因：`KaratsubaMult` 基础情况中 `while(decimalR > 0)` 对 0 不执行
